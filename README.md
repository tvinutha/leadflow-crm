LeadFlow CRM

A modern Full Stack CRM Dashboard built with React that helps manage leads efficiently with authentication, analytics, and export features.

Features
User Authentication (Login / Register)
Dashboard with Lead Analytics (Pie Chart)
Add / Update / Delete Leads
Search & Filter Leads
Lead Status Tracking (New, Contacted, Converted)
Export Leads to Excel file
Dark / Light Mode Toggle
Responsive UI with smooth animations
Real-time UI updates after actions
Tech Stack

Frontend:

React.js
Axios
Recharts
XLSX (Excel export)
File Saver
CSS (Custom styling)

Backend:

Node.js
Express.js
MongoDB
JWT Authentication
